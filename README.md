## epc-ng
#### 依赖
- node >= 6.9.*
- angular-cli  （构建工具，全局模式安装）
- epc-ng-server  （后台数据服务）  
#### 启动
```
  npm install
  npm start  // 项目根目录下运行（开发模式）
```
#### 打包压缩
```
  ng build --prod --aot  // 项目根目录下运行，运行后，将dist文件夹内生成的文件拷贝至服务器相应的目录
```
#### 访问
```
  http://localhost:4200
```
#### 相关
- 图标库：font-awesome
- angular2组件库：primeng
- 集成：requirejs，jquery，svgHotPoint
